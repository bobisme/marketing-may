# Competitive Intelligence

Load when conducting a competitor teardown, mapping a market, identifying gaps, or comparing positioning/pricing across alternatives.

## Contents

- Competitor set (4 groups)
- Research table
- What to inspect
- Gap taxonomy

## Competitor set

Build four groups:

1. **Direct competitors:** same category, same buyer.
2. **Adjacent competitors:** overlapping buyer/problem, different category.
3. **Substitutes:** manual work, spreadsheets, agencies, internal tools, open source, doing nothing.
4. **Aspirational references:** companies with strong positioning, pricing, onboarding, or trust design in adjacent markets.

## What to inspect

- Homepage hero and category noun.
- Pricing page: model, value metric, packages, free trial/freemium, enterprise CTA, guarantees.
- Docs/onboarding: setup complexity, activation path, integrations.
- Reviews: repeated pains, desired outcomes, switching reasons, objections.
- Ads and SEO pages: keywords, use cases, claims, offers.
- Trust pages: security, privacy, compliance, case studies, reviews.
- Support/community: unresolved complaints and feature gaps.

## Gap taxonomy

| Gap type | Meaning | Opportunity |
|---|---|---|
| Segment gap | Competitors speak broadly | Narrow ICP landing page/outbound wedge |
| Proof gap | Claims without evidence | Publish benchmark, case study, transparent demo |
| Pricing gap | Confusing, too expensive, hidden | Simple package, calculator, transparent entry plan |
| UX gap | Setup/onboarding too hard | Concierge onboarding, template, migration tool |
| Channel gap | Competitors ignore a reachable channel | Founder-led content, community, integration, marketplace |
| Trust gap | Privacy/security unclear | Security page, clear policies, compliance-first copy |
| Outcome gap | Feature-heavy messaging | Outcome-specific copy and ROI framing |
