## Experiment design checklist

- [ ] One primary assumption.
- [ ] One primary metric.
- [ ] One guardrail metric.
- [ ] Audience and unit defined.
- [ ] Threshold set before launch.
- [ ] Decision rule set before launch.
- [ ] Instrumentation QA complete.
- [ ] Ethical/compliance review complete.
- [ ] Post-test analysis date set.

## Common thresholds by test type

These are starting points, not universal benchmarks. Calibrate to category, ACV, channel, and baseline.

| Test | Weak | Promising | Strong |
|---|---:|---:|---:|
| Problem interview: repeated urgent pain | <30% | 30–60% | >60% |
| Cold outbound positive reply | <1% | 1–5% | >5% |
| High-intent landing qualified CTA | <2% | 2–8% | >8% |
| Waitlist from broad traffic | <1% | 1–3% | >3%, if ICP-qualified |
| Paid pilot close from qualified call | <10% | 10–30% | >30% |
| Trial activation | <20% | 20–50% | >50% |
| Repeat usage after activation | <20% | 20–40% | >40% |
