## Segment priority scoring

Score 1–5. Use evidence confidence as 0.2–1.0.

| Segment | Pain | Trigger | WTP | Reachability | Differentiation | Proof availability | Sales complexity | Switching cost | Compliance risk | Build burden | Evidence confidence | Score |
|---|---:|---:|---:|---:|---:|---:|---:|---:|---:|---:|---:|---:|
| | | | | | | | | | | | | |

Formula:

```text
Score = ((Pain * Trigger * WTP * Reachability * Differentiation * ProofAvailability) /
         (SalesComplexity * SwitchingCost * ComplianceRisk * BuildBurden)) * EvidenceConfidence
```

## ICP one-liner

```text
For [role/team] at [company type/stage] who [trigger + painful workflow], [product] helps them [outcome] by [mechanism], unlike [alternative], with proof from [evidence].
```

## JTBD story

```text
When [trigger/circumstance], I want to [progress], so I can [functional outcome], feel [emotional outcome], and be seen as [social outcome]. I currently use [alternative], but it fails because [struggle]. I will switch if [switch condition], and I will resist because [anxiety/inertia].
```
