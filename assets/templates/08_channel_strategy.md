## Channel prerequisites

| Channel | Must be true before scaling |
|---|---|
| Paid search | High-intent terms, conversion path, price supports CAC |
| Paid social | Strong creative/proof, retargeting or clear audience, funnel tracking |
| SEO/AEO | Search language exists, content can be differentiated, conversion CTA |
| Outbound | Named account list, trigger, specific offer, compliance process |
| Partnerships | Partner has same ICP and gains clear value |
| Community | Founder can contribute value without spamming |
| Marketplace | Listing can activate users and collect reviews |
| Referrals | Users are retained and know whom to refer |
